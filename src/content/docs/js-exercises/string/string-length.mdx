#### ðŸ’¡ Solution

```js
// example.js
function getStringLength(str) {
    return str.length;
}

// Test Data
console.log(getStringLength('Hello, World!')); // 13
console.log(getStringLength('JavaScript'));    // 10
console.log(getStringLength(''));              // 0
```

#### ðŸ“˜ Explanation

This is a simple function that tells you how many characters are in a piece of text (string). It's like counting letters in a word!


**How It Works:**

	- **Takes a string input** (any text inside quotes like `"Hello"` or `"JavaScript"`)
	- **Uses `.length` property** to count all characters in that string
	- **Returns the count** (the number of characters)


**Examples**

1. `'Hello, World!'` â†’ 13 characters


	- Count: H(1), e(2), l(3), l(4), o(5), ,(6), (space)(7), W(8), o(9), r(10), l(11), d(12), !(13)


2. `'JavaScript'` â†’ 10 characters

	- Count: J(1), a(2), v(3), a(4), S(5), c(6), r(7), i(8), p(9), t(10)

3. `''` (empty string) â†’ 0 characters

	- There's nothing to count!

**Key Notes:**

- It counts EVERYTHING between the quotes:
	- Letters (a-z, A-Z)
	- Numbers (0-9)
	- Spaces ( )
	- Punctuation (!, ?, . etc.)
	- Special characters (@, #, $ etc.)
- An empty string (`''`) has length 0
