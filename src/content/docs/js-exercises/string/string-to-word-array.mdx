```js
// example.js
function stringToWords(str) {
  return str.split(' ');
}

// Test Data
console.log(stringToWords('Hello world! This is JavaScript.')); // ["Hello", "world!", "This", "is", "JavaScript."]
console.log(stringToWords('Split this string into words')); // ["Split", "this", "string", "into", "words"]
```

It's like taking a sentence and cutting it into separate words wherever there are spaces, then putting all the words in a list. âœ‚ï¸ğŸ“

1. **Input:** It takes a string (sentence) like `"Hello world!"`
2. **Splitting:** It uses `split(' ')` to cut the string wherever there's a **space** (`' '`)
3. **Output:** It returns an **array** (list) of the separated words

For Example: `"Hello world! This is JavaScript.":`

1. Finds spaces and splits at each one:

- "Hello" + " " â†’ first word: "Hello"
- "world!" + " " â†’ second word: "world!"
- "This" + " " â†’ third word: "This"
- "is" + " " â†’ fourth word: "is"
- "JavaScript." (no space after) â†’ fifth word: "JavaScript."

2. Returns: `["Hello", "world!", "This", "is", "JavaScript."]`

**Key Points:**

- The space `' '` inside `split(' ')` tells JavaScript where to split
- Punctuation (like "!" and ".") stays with the word it's attached to
- If there are multiple spaces, they'll result in empty strings in the array
